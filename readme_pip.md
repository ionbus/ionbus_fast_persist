# ionbus_utils

A collection of Python utilities for common development tasks including
logging, cryptography, file operations, date/time handling, caching,
pandas operations, subprocess management, and configuration management.

## Installation

```bash
pip install ionbus-fast-persist
```

Or install from source:

```bash
pip install -e .
```


Full [documentation on github](https://github.com/ionbus/ionbus_fast_persist?tab=readme-ov-file#ionbus-fast-persist).

## Requirements

- Python >= 3.9
- See `requirements.txt` for dependencies
